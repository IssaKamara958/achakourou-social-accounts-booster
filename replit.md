# Achakourou Booster AI

AI-powered TikTok/Instagram/Facebook social media management SaaS for African creators and agencies.

## Run & Operate

- `pnpm --filter @workspace/booster-ai run dev` — run the frontend (Vite dev server)
- `pnpm --filter @workspace/api-server run dev` — run the API server (port 5000)
- `pnpm run typecheck` — full typecheck across all packages
- `pnpm run build` — typecheck + build all packages
- Required env: `VITE_SUPABASE_URL`, `VITE_SUPABASE_ANON_KEY` — Supabase credentials for the frontend

## Stack

- pnpm workspaces, Node.js 24, TypeScript 5.9
- Frontend: React + Vite + TanStack Router + TanStack Query
- Backend: Supabase (auth + database)
- UI: shadcn/ui + Tailwind CSS v4 + Radix UI
- AI: Vercel AI SDK + OpenAI-compatible
- Animations: Framer Motion

## Where things live

- `artifacts/booster-ai/` — React + Vite frontend SPA
- `artifacts/booster-ai/src/routes/` — TanStack file-based routes (app.*, index, login, onboarding)
- `artifacts/booster-ai/src/lib/` — supabase, auth, quota, AI gateway, social integrations
- `artifacts/booster-ai/src/components/` — UI components (shadcn/ui + app-sidebar)
- `artifacts/booster-ai/src/routeTree.gen.ts` — auto-generated by TanStack Router (do not edit)
- `lib/api-spec/openapi.yaml` — OpenAPI spec (unused by frontend, Supabase handles backend)

## Architecture decisions

- Frontend-only SaaS backed entirely by Supabase (no custom Express API needed for core features)
- TanStack Router with file-based routing — routes live in `src/routes/*.tsx`
- Tailwind CSS v4 via `@tailwindcss/vite` plugin (no postcss.config.cjs — conflicts with v4)
- Supabase Auth + Supabase DB used directly from the frontend via RLS policies
- Quota system is localStorage-based (daily limits for AI calls, posts, SEO)

## Product

- Landing page with agency pitch
- Auth: sign in / sign up / onboarding flow
- Dashboard with viral score analytics and weekly charts
- Trends Engine: real-time TikTok/Instagram trend discovery
- AI Generator: viral script generation (hook + content + CTA) with Supabase AI gateway
- Social Accounts: OAuth connect for TikTok, Instagram, Facebook
- Publisher: schedule and publish content
- SEO Pages: AI-generated SEO pages
- Client management: multi-client workspace

## Gotchas

- Set `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY` secrets — app will load without them but Supabase calls will fail
- Never add `postcss.config.cjs` to the artifact — it conflicts with `@tailwindcss/vite` (Tailwind v4)
- `routeTree.gen.ts` is generated by TanStack Router — do not edit manually; regenerate with `@tanstack/router-vite-plugin` or by running the dev server
- The app uses TanStack Router (not wouter) — the scaffold's default wouter setup was replaced
